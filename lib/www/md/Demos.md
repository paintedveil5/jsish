## Builtins

If you have downloaded Jsish, it comes with builtin applications and demos.
For example, to use the Jsi debugger with its Web-UI interface use:

``` bash
jsish -D tests/while2.jsi
```

Or try the Sqlite-UI with:

``` bash
jsish -S nw2.db
```
