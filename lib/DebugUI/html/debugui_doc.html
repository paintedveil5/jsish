DebugUI: a debugger for Jsi

Jsi is a scripting language for C developers, and DebugUI is it's web-based script debugger supporting:
  • Breakpoint toggle by double clicking (on whitespace in a line).
  • Command buttons for step and next, moving up and down call stack, etc.
  • Printing/evaluating values by highlighting a variable and clicking Print or Eval.
  • A halt command to stop execution (in place of control-c).
  • Dialogs/navigators for Functions, Breakpoint, Options and the Stack.
The input area allows commands to be entered that support:
  • Shortcut commands using first 1-2 chars, eg. s for step.
  • Empty line repeat of the last command.
  • Up/down arrow to recall previous commands.
Available commands include: 
args      : Show function arguments
break     : Break execution at a named function, line or file
continue  : Continue execution
delete    : Delete one or all breakpoints
disable   : Disable one or all breakpoints
down      : Move down one or more stack levels
enable    : Enable one or all breakpoints
eval      : Evaluate expression in program context
finish    : Run till return of current function
help      : Display command usage
halt      : Stop execution
info      : Info subcommands: bp args locals var 
list      : List file lines args can be: startLine|func numLines file
locals    : Show function local variables
next      : Single-step over function calls
print     : Print value of variable
quit      : Quit debugging current program and exit
step      : Single-step into function calls
tbreak    : Sets a temporary breakpoint
up        : Move up one or more stack levels
where     : Display current location
xeval     : Does eval in debugger context
